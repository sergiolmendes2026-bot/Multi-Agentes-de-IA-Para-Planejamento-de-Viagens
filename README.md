# Multi-Agentes-de-IA-Para-Planejamento-de-Viagens

https://multi-agentes.streamlit.app/


 Multi-Agentes de IA. Ele vai além da simples execução de um modelo, trazendo uma solução completa que integra três ferramentas poderosas.
O CrewAI, responsável pela colaboração entre agentes; o Groq, que garante alta performance na execução de LLMs; e o Tavily, que conecta a aplicação à web para realizar buscas em tempo real.
Nesse sistema, cada agente tem uma função específica: um cuida da pesquisa, outro do planejamento logístico e o último organiza o roteiro final. 
Trabalhando juntos, eles transformam informações simples fornecidas pelo usuário — como destino, duração e interesses — em um itinerário detalhado e pronto para uso.

O projeto também aborda pontos importantes para aplicações em produção, como integração de APIs, uso seguro de variáveis de ambiente,
orquestração de tarefas, automação de decisões, experiência interativa com Streamlit e boas práticas de organização em sistemas Multi-Agentes.
O resultado é um sistema que une raciocínio, busca externa, planejamento e geração de conteúdo, servindo como exemplo real de como criar, 
implantar e escalar aplicações de IA orientadas a agentes em cenários práticos e de alto impacto.


CrewAI:   framework como agente de IA

Groq: Plataforma para acessar os LLMs 

Tavily: Como uma plataforma de busca, para configurar ferramentas para os agentes de IA
